Projects

Wanderlust Full-Stack Project

Wanderlust is a full-stack travel booking application built using the MERN Stack. It allows users to browse, search, and filter various accommodations. The platform includes secure user authentication, authorization (only owners can edit/delete listings), review and rating systems, dynamic image uploading via Cloudinary, and a fully responsive UI. It follows the MVC architecture and real-time updates ensure a smooth user experience. The backend is connected with MongoDB Atlas and frontend and backend are fully deployed on Render.

Tech Stack: Node.js, Express.js, MongoDB, Mapbox, Bootstrap 5, Cloudinary, EJS, JWT Authentication, Passport.js, Joi, Connect-Flash

Priscripto | Appointment Booking System

Priscripto is a feature-rich full-stack web application designed for seamless appointment management. It provides dedicated portals for Users, Doctors, and Admins. Users can register, login, book appointments with doctors, and manage their bookings. Doctors can set available time slots, view upcoming appointments, and manage their schedules. Admins have access to oversee user and doctor activities, verify doctors, and manage the entire system. The application ensures secure authentication using JWT tokens and offers a smooth and responsive user experience across all devices. Real-time updates and robust backend APIs power the system, making it efficient and reliable.The project follows the MVC (Model-View-Controller) architectural pattern for better code organization and scalability.

Tech Stack: MongoDB, Express.js, React.js, Node.js, Cloudinary, React Toastify, Tailwind CSS, Axios, JWT (JSON Web Tokens), Mongoose, Git & GitHub
